$ KERNEL

A from-scratch x86 operating system — booting live in your browser via v86. This is the exact same disk image QEMU runs.

● booting…

What you're watching: our hand-written boot sector loads the kernel off a virtual disk, switches the CPU from 16-bit real mode into 32-bit protected mode, and hands off to C — which writes straight to VGA memory at 0xB8000. No Linux, no libraries underneath. Milestone 1 of 6.
